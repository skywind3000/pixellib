Description
-----------
- High Quality Anti-Alising Vector/Raster Graphics
- Multi-Pixel-Format: RGB, BGR, ARGB, ABGR, RGBA, BGRA
8 / 15 / 16 / 24 / 32 depth
- Light weight 100% C implementation
- SSE2/MMX Graphics Optimized
